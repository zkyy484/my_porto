About Me

Halo! Saya Moch. Dzaky Musyaddad biasa dipanggil Dzaky. Saat ini saya sedang menempuh studi di Politeknik Negeri Banyuwangi, Jurusan Bisnis dan Informatika, Program Studi Teknik Rekayasa Perangkat Lunak.

Ketertarikan saya pada dunia pemrograman dimulai sejak masa kuliah, khususnya dalam pengembangan website. Sejak saat itu, saya terus belajar dan mengembangkan kemampuan di bidang web development.

Saat ini, saya fokus mengembangkan aplikasi web modern dengan memanfaatkan berbagai teknologi seperti Laravel, JavaScript, serta pengelolaan database. Saya senang membangun website yang tidak hanya berfungsi dengan baik, tetapi juga memiliki desain yang menarik, responsif, dan nyaman digunakan.
